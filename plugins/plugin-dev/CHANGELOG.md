# Changelog — plugin-dev

All notable changes to this plugin. Format: [Keep a Changelog](https://keepachangelog.com), semver.

## [1.0.0] - 2026-07-03

### Added
- `/new-plugin` — scaffold a new marketplace plugin.
- `/vendor-plugin` — import a public GitHub plugin at a pinned commit with a license gate.
- `/validate-plugin` — run the CI validation suite locally.
- `/submit-plugin` — one-shot submission: branch, version bump, changelog, validation, commit, and PR handled for you.
- `plugin-authoring` skill — house rules: risk tiers, versioning, changelog format, what CI rejects.
